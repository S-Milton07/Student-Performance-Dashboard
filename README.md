# 🎓 Student Performance Dashboard

A Full Stack Machine Learning Project built using **React, Flask, Pandas, and Scikit-Learn**.

This project helps analyze student performance by uploading a CSV file, viewing statistics, searching student records, and predicting student results using a Machine Learning model.

---

# 🚀 Features

- 📂 Upload Student CSV File
- 📊 View Student Statistics
- 👨‍🎓 Display Student Records
- 🔍 Search Student by Name
- 📈 Grade Distribution
- 🤖 Machine Learning Prediction
- ✅ Pass / Fail Prediction
- 🎨 Responsive Dashboard UI

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/S-Milton07/Student-Performance-Dashboard.git
```

---

## Backend Setup

```bash
cd backend
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Environment

Windows

```bash
venv\Scripts\activate
```

Install Packages

```bash
pip install -r requirements.txt
```

Run Flask

```bash
python app.py
```

---

## Frontend Setup

```bash
cd frontend
```

Install Packages

```bash
npm install
```

Run React

```bash
npm run dev
```

# 🤖 Machine Learning Model

The prediction model is trained using the following student attributes:

- Age
- Attendance
- Study Hours
- Assignment Marks
- Internal Marks

Output

- ✅ PASS
- ❌ FAIL

Model Used:

- Decision Tree Classifier (Scikit-Learn)
